# Linux Cheatsheet

## Navegação
pwd
ls -la
cd pasta
cd ..

## Arquivos
touch file
cp a b
mv a b
rm file
rm -rf pasta

## Permissões
chmod 600 file
chmod 700 pasta
chmod 755 script
chown user:user file

## Pacotes
sudo apt update
sudo apt upgrade
sudo apt install pacote
